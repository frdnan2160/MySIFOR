# MySIFOR - Panduan Lengkap 

README ini khusus untuk siswa pengguna aplikasi MySIFOR.
Tujuannya supaya kamu bisa langsung paham cara pakai aplikasi dari awal sampai harian.

## 1. Fungsi MySIFOR untuk Siswa

Dengan MySIFOR, kamu bisa:
- Melihat ringkasan kegiatan sekolah hari ini.
- Melihat jadwal pelajaran.
- Mengecek tugas dan deadline.
- Mengecek ujian terdekat.
- Chat dengan teman satu jurusan, petugas, atau pihak administrator.
- Menerima notifikasi penting (chat, informasi, pengingat waktu tertentu).

## 2. Persiapan Sebelum Pakai

Sebelum login, pastikan:
- Kamu sudah punya akun (username + password) dari administrator.
- Internet aktif (Wi-Fi atau data seluler).
- Izin notifikasi aplikasi diaktifkan.

## 3. Cara Install Aplikasi (Android)

1. Download file APK versi terbaru.
2. Buka file APK, lalu tekan `Install`.
3. Jika Android memblokir, aktifkan izin `Install unknown apps` untuk aplikasi yang kamu pakai download.
4. Setelah instal selesai, buka aplikasi MySIFOR.

## 4. Cara Login

1. Masukkan `Username`.
2. Masukkan `Password`.
3. Tekan tombol `Masuk`.

Jika gagal login:
- Cek lagi huruf besar/kecil username dan password.
- Pastikan internet stabil.
- Jika tetap gagal, minta reset akun ke pengelola akun sekolah.

## 5. Cara Baca Dashboard Siswa

Setelah login, halaman dashboard menampilkan ringkasan penting:
- Jadwal saat ini: pelajaran yang sedang/sudah berlangsung.
- Status tugas: daftar tugas aktif dan deadline terdekat.
- Ujian terdekat: info jadwal ujian yang akan datang.
- Jadwal shalat harian: jam shalat pada hari itu.

Tips:
- Buka dashboard setiap pagi sebelum kelas.
- Cek kartu tugas minimal 2 kali sehari (siang dan malam).

## 6. Fitur Chat untuk Siswa

Di menu pesan/chat, kamu bisa:
- Melihat daftar percakapan.
- Cari chat berdasarkan username.
- Buka chat langsung dan kirim pesan.

Perilaku notifikasi chat:
- Notifikasi dikelompokkan per pengirim/ruang chat.
- Jika ada beberapa pesan dari pengirim yang sama, preview akan menampilkan riwayat pesan terbaru.
- Tekan notifikasi untuk langsung masuk ke ruang chat terkait.

## 7. Notifikasi Adzan di Aplikasi

Pengaturan audio adzan:
- Subuh: memakai audio adzan khusus Subuh.
- Dzuhur, Ashar, Maghrib, Isya: memakai audio adzan reguler.

Kalau suara tidak terdengar:
- Naikkan volume media/alarm.
- Pastikan mode senyap tidak aktif.
- Cek izin notifikasi tetap aktif.

## 8. Cara Update Aplikasi

Saat versi baru tersedia, aplikasi akan menampilkan popup update.

Pilihan update:
- `Update Sekarang`: APK diunduh otomatis di latar belakang, lalu kamu lanjut instal.
- `Nanti`: update ditunda (jika update tidak wajib).

Disarankan selalu update agar:
- Bug lama tidak muncul lagi.
- Fitur terbaru langsung aktif.
- Performa aplikasi lebih stabil.

## 9. Troubleshooting Cepat

Notifikasi tidak masuk:
1. Aktifkan izin notifikasi MySIFOR.
2. Nonaktifkan pembatasan baterai untuk MySIFOR.
3. Pastikan internet aktif.
4. Restart HP jika perlu.

Chat loading lama atau macet:
1. Ganti jaringan (dari data ke Wi-Fi atau sebaliknya).
2. Tutup paksa aplikasi lalu buka ulang.
3. Logout lalu login kembali.
4. Pastikan aplikasi sudah versi terbaru.

Aplikasi terasa lambat:
1. Tutup aplikasi lain yang berat.
2. Sediakan ruang penyimpanan kosong.
3. Update ke versi terbaru.

## 10. Bantuan

Jika masih ada kendala akun, data pelajaran, tugas, atau chat:
- Hubungi administrator yang mengelola akun MySIFOR.
