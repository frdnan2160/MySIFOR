# MySIFOR - Panduan Pengguna Lengkap

MySIFOR adalah aplikasi informasi Forum Kesiswaan TJKT berbasis Android.
Panduan ini ditujukan untuk semua pengguna aplikasi:
- `Siswa`
- `Sekretaris` (Special Role)
- `Bendahara` (Special Role)

Catatan penting:
- Pengelolaan sistem, akun, dan konfigurasi aplikasi **hanya** dilakukan oleh `Administrator`.
- Jika ada kendala akun, role, atau data yang tidak sesuai, laporkan ke `Administrator`.

## 1. Tujuan Aplikasi

MySIFOR membantu pengguna untuk:
- Melihat informasi harian secara cepat.
- Memantau jadwal, tugas, ujian, dan agenda.
- Berkomunikasi lewat fitur chat.
- Menerima notifikasi penting.
- Menerima pengingat waktu shalat sesuai pengaturan aplikasi.

## 2. Ringkasan Hak Akses Role

### 2.1 Siswa

Siswa umumnya dapat:
- Melihat dashboard siswa.
- Melihat jadwal, tugas, ujian, dan ringkasan aktivitas.
- Menggunakan chat untuk komunikasi sekolah.
- Menerima notifikasi aplikasi.

### 2.2 Sekretaris (Special Role)

Sekretaris umumnya dapat:
- Mengakses menu administrasi sekretariat yang diizinkan.
- Melihat ringkasan data administrasi.
- Berkomunikasi via chat untuk koordinasi.
- Menerima notifikasi terkait tugas sekretariat.

### 2.3 Bendahara (Special Role)

Bendahara umumnya dapat:
- Mengakses menu keuangan yang diizinkan.
- Memantau data keuangan sesuai kewenangan.
- Berkomunikasi via chat untuk koordinasi keuangan.
- Menerima notifikasi terkait data/transaksi keuangan.


## 3. Persiapan Sebelum Menggunakan Aplikasi

Sebelum install dan login, pastikan:
1. Kamu sudah menerima `username` dan `password` dari Administrator.
2. HP Android punya koneksi internet stabil.
3. Ruang penyimpanan cukup untuk instalasi dan update APK.
4. Izin notifikasi untuk MySIFOR nanti diaktifkan.

## 4. Cara Install MySIFOR (Android)

1. Download APK versi terbaru dari link resmi yang dibagikan sekolah.
2. Buka file APK.
3. Jika muncul peringatan keamanan, aktifkan izin instalasi dari sumber tersebut (`Install unknown apps`).
4. Lanjutkan proses instalasi sampai selesai.
5. Buka aplikasi MySIFOR.

Tips:
- Selalu gunakan APK terbaru agar performa dan stabilitas lebih baik.
- Jangan install APK dari sumber yang tidak jelas.

## 5. Login Pertama dan Login Harian

1. Buka aplikasi MySIFOR.
2. Isi `Username`.
3. Isi `Password`.
4. Tekan tombol `Masuk`.

Jika login berhasil:
- Sistem akan membuka dashboard sesuai role akunmu.

Jika login gagal:
1. Pastikan `username/password` benar (perhatikan huruf besar-kecil).
2. Pastikan internet aktif.
3. Coba tutup aplikasi lalu buka ulang.
4. Jika tetap gagal, minta bantuan Administrator untuk cek akun.

## 6. Panduan Dashboard

Dashboard adalah halaman utama setelah login.
Isi dashboard bisa berbeda tergantung role.

### 6.1 Dashboard Siswa

Umumnya menampilkan:
- Ringkasan pelajaran hari ini.
- Status tugas aktif dan deadline terdekat.
- Ujian terdekat.
- Jadwal shalat harian.

Rutin yang disarankan:
1. Pagi: cek jadwal pelajaran dan tugas.
2. Siang: cek pembaruan tugas/ujian.
3. Malam: pastikan tidak ada deadline terlewat.

### 6.2 Dashboard Sekretaris

Umumnya menampilkan:
- Ringkasan administrasi sesuai kewenangan.
- Akses cepat ke menu sekretariat.
- Notifikasi informasi internal yang relevan.

Rutin yang disarankan:
1. input data kelas setiap informasi terbaru.
2. Tindak lanjuti chat koordinasi dari pihak sekolah.
3. Pastikan data administrasi terbaru sudah sinkron.

### 6.3 Dashboard Bendahara

Umumnya menampilkan:
- Ringkasan data keuangan sesuai kewenangan.
- Akses cepat ke menu bendahara.
- Notifikasi terkait transaksi atau pembaruan data.

Rutin yang disarankan:
1. Cek ringkasan data keuangan di awal hari.
2. Tinjau notifikasi transaksi/perubahan data.
3. Lakukan verifikasi data sebelum akhir hari.

## 7. Panduan Fitur Chat

Fitur chat dipakai untuk komunikasi antar pengguna sesuai akses.

### 7.1 Di Halaman Daftar Chat

Kamu bisa:
- Melihat daftar percakapan.
- Melakukan pencarian chat berdasarkan nama/identitas.
- Membuka ruang chat yang dipilih.

Jika tersedia tab `Chat` dan `Arsip`:
- `Chat` berisi percakapan aktif.
- `Arsip` berisi percakapan yang diarsipkan.

### 7.2 Di Dalam Ruang Chat

Kamu bisa:
- Membaca riwayat pesan.
- Mengirim pesan baru.
- Menerima pesan balasan real-time (tergantung koneksi).

Tips chat:
- Gunakan pesan yang jelas dan singkat.
- Cek ulang pesan sebelum kirim.
- Pastikan koneksi stabil saat kirim pesan penting.

### 7.3 Notifikasi Chat

Perilaku notifikasi chat:
- Notifikasi dikelompokkan per pengirim/ruang chat.
- Jika ada banyak pesan dari pengirim yang sama, preview akan menampilkan beberapa pesan terbaru.
- Menekan notifikasi akan membuka chat terkait.

## 8. Notifikasi Waktu Shalat dan Audio Adzan

Pengaturan audio:
- `Subuh` memakai audio adzan khusus Subuh.
- `Dzuhur`, `Ashar`, `Maghrib`, `Isya` memakai audio adzan reguler.

Agar notifikasi adzan berjalan baik:
1. Aktifkan izin notifikasi aplikasi.
2. Pastikan volume media/alarm tidak nol.
3. Nonaktifkan mode senyap jika ingin suara terdengar.
4. Matikan pembatasan baterai ekstrem untuk MySIFOR.

## 9. Cara Update Aplikasi

Saat versi baru tersedia, aplikasi menampilkan popup update.

Opsi update:
- `Update Sekarang`: aplikasi mengunduh APK di latar belakang, lalu pengguna melanjutkan instalasi.
- `Nanti`: menunda update (jika update tidak wajib).

Langkah aman setelah update:
1. Selesaikan proses instal APK sampai tuntas.
2. Buka ulang MySIFOR.
3. Cek versi aplikasi jika diperlukan.
4. Login kembali bila diminta.

## 10. Keamanan Akun Pengguna

Agar akun aman:
1. Jangan bagikan password ke orang lain.
2. Ganti password lewat Administrator jika dicurigai bocor.
3. Logout jika memakai perangkat pinjaman.
4. Hindari login di perangkat yang tidak kamu percaya.

## 11. Troubleshooting Lengkap

### 11.1 Notifikasi Tidak Muncul

1. Cek izin notifikasi MySIFOR (harus `Allow`).
2. Cek pengaturan baterai, pastikan MySIFOR tidak dibatasi.
3. Cek koneksi internet.
4. Restart HP.
5. Buka aplikasi sekali untuk memastikan sinkronisasi aktif.

### 11.2 Notifikasi Ada, Tapi Tidak Bersuara

1. Naikkan volume media/alarm.
2. Nonaktifkan mode senyap dan `Do Not Disturb`.
3. Cek kanal notifikasi aplikasi, pastikan suara aktif.
4. Coba kirim pesan uji untuk memastikan suara notifikasi muncul.

### 11.3 Chat Loading Lama atau Stuck

1. Pindah jaringan internet (Wi-Fi <-> data seluler).
2. Tutup paksa aplikasi, lalu buka lagi.
3. Logout, lalu login ulang.
4. Update aplikasi ke versi terbaru.
5. Jika tetap terjadi, laporkan ke Administrator dengan detail waktu kejadian.

### 11.4 Gagal Update APK

1. Pastikan ruang penyimpanan cukup.
2. Pastikan izin instalasi dari sumber terkait sudah aktif.
3. Hapus file APK lama yang rusak, lalu download ulang.
4. Coba instal ulang melalui file APK terbaru.

### 11.5 Aplikasi Terasa Lambat

1. Tutup aplikasi lain yang berat.
2. Sisakan ruang penyimpanan internal.
3. Restart perangkat.
4. Gunakan versi MySIFOR terbaru.

## 12. FAQ Singkat

### Apakah semua pengguna punya menu yang sama?
Tidak. Menu mengikuti role akun (`Siswa`, `Sekretaris`, `Bendahara`) dan hak akses yang diberikan Administrator.

### Siapa yang bisa ubah role akun?
Hanya `Administrator`.

### Jika data tidak sesuai, harus lapor ke siapa?
Lapor ke `Administrator` sekolah dengan detail masalah.

### Apakah update aplikasi wajib?
Tergantung kebijakan versi. Jika update ditandai wajib, pengguna harus update agar bisa lanjut memakai aplikasi dengan normal.

## 13. Bantuan

Jika kamu mengalami masalah:
1. Catat masalahnya (fitur mana, jam berapa, gejalanya apa).
2. Sertakan screenshot jika ada.
3. Kirim laporan ke `Administrator` sekolah.

Dengan informasi yang lengkap, proses perbaikan biasanya jauh lebih cepat.
